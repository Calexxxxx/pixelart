Pixel Art Maker Info

  • Gird Settings

    • Width - amount of columns.

    • Height - amount of rows.

    • Create Grid - this will create the grid.

    • Reset Grid - this will reset the grid to appart basic size is W x H - 40 x 30.

  • Color Picker

    • Selected Color - this is the color you picked and will be used on the grid.

    • Hover Color - this is the color where the crosshair currently is positioned.

    • Pick a color - with a left mouse click you will select the color.

  • Grid

    • Single Click - this will color the square with the selected color.

    • Double Click - this will remove the color from the square.
